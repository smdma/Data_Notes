## 常用统计函数

1. `count()` 统计个数 NaN 和 None 除外
2. `describe()` 一次性输出多个统计指标
3. `min()` 最小值
4. `max()` 最大值
5. `sum()` 求和
6. `mean()` 平均值
7. `median()` 中位数
8. `var()` 方差
9. `std()` 标准差
10. `argmin()` 统计最小值的索引位置
11. `argmax()` 统计最大值的索引位置
12. `idxmin()` 统计最小值的索引值
13. `idxmax()` 统计最大值的索引值
